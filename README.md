## 📱 应用简介

**拾色师**是一款功能强大的配色设计工具，专为设计师、开发者和配色爱好者打造。从图片中智能提取配色，到从零创作配色方案，再到系统化学习配色理论，拾色师为您提供完整的配色设计解决方案。

### ✨ 核心特性

- 🎯 **智能提取** - 双算法支持（K-Means + 中位切分），精准提取图片主色调
- 🖌️ **自由创作** - 配色实验室，从零创建配色，支持 HSL 精确调整
- 📊 **专业分析** - 色彩情感分析、WCAG 可访问性检查、适用场景推荐
- ⚖️ **科学对比** - 多方案并排对比，数据化分析相似度和差异
- 📚 **系统学习** - 6 大配色理论 + 3 类实战案例，即学即用
- 💾 **本地存储** - 完全单机运行，数据本地保存，保护隐私
- 🎨 **灵感广场** - 30+ 精选配色方案，一键保存使用

## 🚀 核心功能

### 1️⃣ 智能色彩提取

**从图片中提取美丽配色**

- **双算法支持**
  - K-Means 聚类算法 - 精准识别主色调
  - 中位切分算法 - 快速提取代表色
  
- **双模式取色**
  - 自动提取 - 一键提取 6 种主色调
  - 手动取色 - 点击图片精确选色（最多 8 个）

- **完整色彩信息**
  - HEX 格式（#RRGGBB）
  - RGB 格式（rgb(r, g, b)）
  - 颜色占比百分比
  - 一键复制到剪贴板

### 2️⃣ 配色实验室 🔬

**从零创建专属配色方案**

- **自由创作** - 不依赖图片，随时创作配色
- **精确调整** - HSL 色彩空间调整
  - 色相（Hue）：0-360°
  - 饱和度（Saturation）：0-100%
  - 亮度（Lightness）：0-100%
  
- **快速生成**
  - 随机颜色 - 激发灵感
  - 互补色 - 强烈对比
  - 类似色 - 和谐统一
  
- **实时预览** - 所见即所得，支持保存

### 3️⃣ 配色分析仪表盘 📊

**专业的色彩情感与可访问性分析**

- **色彩情感分析**
  - 温暖度评分（0-100）- 冷暖色调倾向
  - 活力度评分（0-100）- 视觉冲击力
  - 专业度评分（0-100）- 商务正式程度
  - 和谐度评分（0-100）- 色彩协调性

- **情感标签识别**
  - 温暖/冷静/活力/优雅/现代/复古等

- **适用场景推荐**
  - 品牌设计/网页设计/UI 界面/海报设计等

- **WCAG 可访问性检查**
  - 对比度比值计算
  - WCAG AA 标准（4.5:1）
  - WCAG AAA 标准（7:1）
  - 等级评估（优秀/良好/一般/较差）

### 4️⃣ 配色对比工具 ⚖️

**数据化对比，科学决策**

- **多方案对比** - 同时对比 2-3 个配色方案
- **并排预览** - 直观查看配色差异
- **详细数据分析**
  - 颜色数量对比
  - 平均亮度对比
  - 平均饱和度对比
  - 色相范围对比
  - 相似度分析（2 个方案时）

## 📖 使用指南

### 快速开始

1. **提取配色**
   - 点击"选择图片"按钮
   - 从相册选择图片
   - 选择提取算法（K-Means 或中位切分）
   - 等待提取完成
   - 查看配色结果

2. **创作配色**
   - 进入"实验室"标签
   - 点击"添加颜色"
   - 使用 HSL 滑块调整颜色
   - 或使用快速生成功能
   - 保存配色方案

3. **分析配色**
   - 在配色库中选择方案
   - 点击"分析"按钮
   - 查看情感分析和可访问性报告

4. **学习配色**
   - 进入"学习"标签
   - 选择感兴趣的教程
   - 阅读理论知识
   - 查看配色示例
