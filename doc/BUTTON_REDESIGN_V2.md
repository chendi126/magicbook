## 🎯 **按钮设计规范**

### 📏 **尺寸**
- **直径**: 36px
- **类型**: `ButtonType.Circle`
- **间距**: 4px
- **图标大小**: 18px

### 🎨 **颜色方案**

| 按钮 | 图标 | 背景色 | 说明 |
|------|------|--------|------|
| 预览 | 👁 | #F0F0F0 | 浅灰色 |
| 调整 | 🎨 | #F0F0F0 | 浅灰色 |
| 生成 | ✨ | #F0F0F0 | 浅灰色 |
| 导出 | 📤 | #F0F0F0 | 浅灰色 |
| 删除 | 🗑 | #FFEBEE | 浅红色（警告色） |

**设计原则**：
- 统一使用浅灰色背景，低调不抢眼
- 删除按钮使用浅红色，突出警告性
- 图标本身已有颜色，无需额外着色

## 🔄 **设计对比**

### ❌ **V1 设计问题**
```
第一行: [🎨 调整] [👁️ 预览] [✨ 生成]
第二行: [📤 导出] [🗑️ 删除]
```
- ❌ 两行布局占用空间大
- ❌ 图标+文字冗余
- ❌ 彩色按钮太花哨
- ❌ 不符合鸿蒙原生风格

### ✅ **V2 设计优势**
```
单行: [👁] [🎨] [✨] [📤] [🗑]
```
- ✅ 单行布局，节省空间
- ✅ 纯图标，简洁明了
- ✅ 统一浅色背景，低调优雅
- ✅ 符合鸿蒙原生设计规范
- ✅ 圆形按钮，现代感强

## 🔧 **技术实现**

### 代码示例

```typescript
Row() {
  // 左侧：名称和时间
  Column() {
    Text(scheme.name)
      .fontSize(16)
      .fontWeight(FontWeight.Medium)
    
    Text(this.formatDate(scheme.createTime))
      .fontSize(12)
      .fontColor('#999999')
  }
  .layoutWeight(1)
  .alignItems(HorizontalAlign.Start)

  // 右侧：操作按钮
  Row({ space: 4 }) {
    // 预览
    Button({ type: ButtonType.Circle }) {
      Text('👁').fontSize(18)
    }
    .width(36)
    .height(36)
    .backgroundColor('#F0F0F0')
    .onClick(() => { this.showPreviewDialog(scheme); })

    // 调整
    Button({ type: ButtonType.Circle }) {
      Text('🎨').fontSize(18)
    }
    .width(36)
    .height(36)
    .backgroundColor('#F0F0F0')
    .onClick(() => { this.showAdjustDialog(scheme); })

    // ... 其他按钮
  }
}
```
