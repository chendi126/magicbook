# 🎨 灵感广场功能指南

## 🎯 功能概述

灵感广场是一个内置的精选配色方案库，为用户提供专业的配色灵感和参考。用户可以浏览、预览和保存喜欢的配色方案到自己的配色库。

---

## ✨ 核心功能

### 1. 精选配色方案 ✅
- **30 个精选配色** - 涵盖 7 大分类
- **专业设计** - 每个配色都经过精心挑选
- **详细描述** - 每个方案都有名称和描述

### 2. 分类筛选 ✅
- **7 大分类**：
  - 🌿 **自然** - 森林、海洋、日落等自然色
  - 💻 **科技** - 赛博朋克、霓虹、数字蓝等
  - 📻 **复古** - 老照片、复古海报等
  - 🏢 **现代** - 极简、莫兰迪、北欧风等
  - ☀️ **温暖** - 暖阳、焦糖、蜜桃等
  - ❄️ **冷色** - 冰川、薄荷、深海等
  - 🎨 **中性** - 大地、水泥、咖啡等

### 3. 一键保存 ✅
- **快速保存** - 点击"保存"按钮即可添加到配色库
- **自动分组** - 保存的配色自动归类到"灵感广场"分组
- **保留分类** - 原始分类作为标签保存

### 4. 网格布局 ✅
- **双列网格** - 充分利用屏幕空间
- **卡片设计** - 美观的卡片式展示
- **颜色预览** - 直观显示所有颜色

## 🚀 技术实现

### 数据结构
```typescript
export interface InspirationScheme {
  id: string;           // 唯一标识
  name: string;         // 方案名称
  category: string;     // 分类
  colors: string[];     // 颜色数组
  description: string;  // 描述
}
```

### 核心功能
- **分类筛选** - 使用 `filter()` 方法筛选
- **网格布局** - 使用 `GridRow` 和 `GridCol`
- **保存功能** - 调用 `DatabaseManager.saveColorScheme()`
- **Toast 提示** - 使用 `promptAction.showToast()`
