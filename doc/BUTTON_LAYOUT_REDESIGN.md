## 🔧 **技术实现**

### `buildActionButton()` 方法

```typescript
@Builder
buildActionButton(icon: string, label: string, color: string, onClick: () => void) {
  Column() {
    Text(icon)
      .fontSize(22)
      .margin({ bottom: 2 })

    Text(label)
      .fontSize(11)
      .fontColor('#FFFFFF')
      .fontWeight(FontWeight.Medium)
  }
  .width(100)
  .height(64)
  .backgroundColor(color)
  .borderRadius(10)
  .justifyContent(FlexAlign.Center)
  .shadow({
    radius: 8,
    color: color + '40',  // 40% 透明度
    offsetX: 0,
    offsetY: 2
  })
  .onClick(onClick)
}
```

### 使用示例

```typescript
// 第一行
Row() {
  this.buildActionButton('🎨', '调整', '#FF9500', () => {
    this.showAdjustDialog(scheme);
  })

  Blank().width(8)

  this.buildActionButton('👁️', '预览', '#5856D6', () => {
    this.showPreviewDialog(scheme);
  })

  Blank().width(8)

  this.buildActionButton('✨', '生成', '#34C759', () => {
    this.showGeneratorDialog(scheme);
  })
}
```

## 🎯 **设计原则**

### 1. **视觉层次**
- 主要功能在上，次要功能在下
- 使用颜色区分不同功能

### 2. **易用性**
- 按钮足够大（100x64），易于点击
- 图标+文字，降低认知负担

### 3. **一致性**
- 所有按钮统一尺寸
- 统一的圆角和阴影

### 4. **美观性**
- 柔和的阴影效果
- 合理的间距
- 和谐的配色
