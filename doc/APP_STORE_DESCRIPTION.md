## 📝 应用描述（详细版）

### 应用介绍

拾色师是一款基于 HarmonyOS 开发的专业配色设计工具，为设计师、开发者和配色爱好者提供完整的配色解决方案。

### 核心功能

**🎯 智能色彩提取**
- 双算法支持：K-Means 聚类 + 中位切分算法
- 双模式取色：自动提取 + 手动点选
- 完整色彩信息：HEX、RGB 格式，一键复制

**🔬 配色实验室**
- 从零创建配色方案，不依赖图片
- HSL 色彩空间精确调整（色相/饱和度/亮度）
- 快速生成：随机色、互补色、类似色
- 实时预览，支持保存

**📊 配色分析仪表盘**
- 色彩情感分析：温暖度、活力度、专业度、和谐度
- 情感标签识别：温暖、冷静、活力、优雅等
- 适用场景推荐：品牌设计、网页设计、UI 界面等
- WCAG 可访问性检查：符合国际标准

**⚖️ 配色对比工具**
- 同时对比 2-3 个配色方案
- 详细数据分析：亮度、饱和度、色相范围
- 相似度计算，科学决策

**📚 配色学习中心**
- 6 大配色理论：色彩基础、互补色、类似色、三角配色、单色配色、四角配色
- 3 类实战案例：企业品牌、运动健身、环保健康
- 可视化示例，即学即用

**💾 配色库管理**
- 智能分类：12 个预设标签 + 自定义标签
- 高效搜索：名称、标签、收藏筛选
- 多格式导出：CSS、SCSS、JSON、Tailwind、ArkTS

**🌟 灵感广场**
- 30+ 精选配色方案
- 按风格分类，一键保存

### 应用特色

✅ **专业算法** - 基于色彩理论和国际标准
